# Docker Craft

A simple Docker lab submission and review system.

## Usage

1. Create a new lab by submitting a title, difficulty, estimated time, and markdown content.
2. The lab will be added to a pending queue and flagged for admin review.
3. Admins can approve or reject submissions via a simple UI.
4. Approved labs will appear in the public list within 5 minutes.
5. Submitters will receive an email notification of the decision.
